# PRD (Product Requirements Document)
# 쿠팡 소싱 도우미 - AI 기반 트렌드 예측 플랫폼

---

## 1. 제품 개요

### 1.1 제품명
**쿠팡 소싱 도우미 (Coupang Sourcing Assistant)**

### 1.2 제품 목적
온라인 쇼핑몰 셀러(특히 쿠팡 판매자)가 데이터 기반으로 유망 상품을 발굴하고, 최적의 제품명/키워드를 추천받아 경쟁력 있는 상품 소싱을 할 수 있도록 지원하는 AI 플랫폼

### 1.3 타겟 사용자
- 쿠팡 신규/기존 셀러
- 스마트스토어 → 쿠팡 확장 셀러
- 해외 소싱(알리/타오바오) 셀러
- 소규모 1인 셀러

### 1.4 핵심 가치 제안
| 가치 | 설명 |
|------|------|
| 데이터 기반 의사결정 | 감이 아닌 데이터로 상품 선정 |
| 미래 트렌드 예측 | LSTM + Claude AI 이중 분석 |
| 틈새 키워드 발굴 | 경쟁 회피 가능한 롱테일 키워드 추천 |
| 원스톱 소싱 | 분석 → 소싱처 연결 → 마진 계산 한 번에 |

## 4. 사용자 시나리오

### 시나리오 1: 신규 상품 발굴
```
1. 사용자가 기간(2020.01~2025.11)과 카테고리(생활용품) 선택
2. 시스템이 네이버 데이터랩에서 트렌드 데이터 수집
3. LSTM 모델이 향후 6개월 검색량 예측
4. Claude AI가 TOP 10 유망 키워드 선정 및 이유 설명
5. 사용자가 "실리콘 주걱" 키워드 선택
6. 시스템이 쿠팡 경쟁강도 분석 결과 제공
7. 틈새 키워드 "실리콘 주걱 내열 세트" 추천
8. 알리바바에서 해당 제품 검색 결과 표시
9. 마진 계산기로 예상 수익 확인
```

### 시나리오 2: 기존 상품 리뉴얼
```
1. 사용자가 기존 판매 상품 "에코백" 입력
2. 시스템이 현재 경쟁 상황 분석
3. 더 유망한 틈새 키워드 추천
4. 제품명 최적화 제안
```

## 6. 제약 조건

### 6.1 기술적 제약
- 네이버 데이터랩 API 호출 제한 (일 1,000회)
- 쿠팡 공식 API 미제공 (크롤링 필요, 법적 리스크)
- LSTM 학습 시간 (초기 로딩 지연 가능)

### 6.2 비즈니스 제약
- 의류 카테고리 제외
- 부피가 큰 상품 제외 (택배 배송 가능 크기)

### 6.3 법적 제약
- 네이버/쿠팡 이용약관 준수
- 개인정보 미수집
